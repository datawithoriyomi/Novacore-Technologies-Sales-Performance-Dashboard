## Project Overview
Novacore Technologies is a Pan-African tech-driven company founded in 2019, operating across Marketplace, Logistics, and Payment Services. As the company expands across multiple African regions, leadership requires deeper insights into operational performance using historical sales data.

This project focuses on analyzing transactional data and developing an interactive Excel dashboard to support data-driven decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning & Transformation
- Data Visualization
- Business Analysis

## Dashboard Features
The dashboard provides insights into key business performance metrics including:

- Count of Sales Records
- Total Sales
- Total Profit
- Total Discount
- Total Quantity Sold
- Monthly Sales Trends
- Sales by Category
- Profit by Segment
- Top 5 Most Discounted Sub-Categories
- Bottom 5 Products with Lowest Profit
- Ship Mode Filter (Interactive Slicer)

### Bonus Analysis
- Top 5 Most Profitable Customers per Year grouped by Region

## Business Insights
The dashboard enables stakeholders to:

- Track revenue performance over time
- Identify profitable customer segments
- Detect products with low profitability
- Evaluate the impact of discounts on profit
- Monitor regional sales performance
- Support strategic decision-making
